## 🌍 World Clock Plugin

Reloj mundial que muestra la hora actual en diferentes zonas horarias.

### Características

- ⏰ Sincronización en tiempo real
- 🌐 12 zonas horarias predefinidas
- ✅ Selecciona las zonas que quieras ver
- 🎨 Interfaz limpia y moderna

### Zonas disponibles

- **UTC** - Hora Universal Coordinada
- **EST/CST/MST/PST** - Zonas de USA
- **CET/EET** - Zonas de Europa
- **IST** - India
- **JST** - Japón
- **AEST** - Australia
- **NZST** - Nueva Zelanda

### Instalación

1. Ve a `🏪 Marketplace` en NetHUB
2. Busca "World Clock"
3. Haz clic en "Instalar"
4. ¡Listo! El módulo se cargará automáticamente

### Uso

1. Selecciona las zonas horarias que quieras monitorear
2. Las horas se actualizarán automáticamente cada segundo
3. Puedes cambiar de zonas en cualquier momento
